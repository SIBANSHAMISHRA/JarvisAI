# JarvisAI 0.1

A simple voice assistant built with Python.  
Features:
- Speech recognition
- Text-to-speech
- Open folders
- Tell time

## Installation
1. Clone the repo
2. Install requirements:
3. Run: pip install -r requirements.txt
